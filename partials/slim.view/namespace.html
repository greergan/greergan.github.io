Namespace

A variable supplied when a new view is created. This variable is prepended to all relative paths found inside of include statements

Used when creating a new SlimView object.

import * as slim from "https://greergan.github.io"/slim.view/index.ts";
const view:slim.view.SlimView = new slim.view.SlimView(namespace_string);

The "file" protocol.

file:///path/to/website/root

The "http" protocol.

http://greergan.github.io

The "https" protocol.

https://greergan.github.io
