Filters

Slim.View uses the Slim.Filter JSON filtering engine.

All filters take the following form. {% foreach model="equal(users, profession, musician)" view="partials/user.html" %}

{% foreach model="equal(users, profession, musician)" view="partials/user.html" %}

{% foreach model="not_equal(users, profession, musician)" view="partials/user.html" %}

{% foreach model="not_equal(users, profession, musician)" view="partials/user.html" %}

{% foreach model="greater_than(users, first_name, Ian)" view="partials/user.html" %}

{% foreach model="greater_than(users, first_name, Ian)" view="partials/user.html" %}

{% foreach model="less_than_equal(users, last_name, Ford)" view="partials/user.html" %}

{% foreach model="less_than_equal(users, last_name, Ford)" view="partials/user.html" %}
