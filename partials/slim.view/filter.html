Filters

Slim.View uses the Slim.Filter JSON filtering engine.

All filters take the following form. {% foreach model="page.users" filter="equal(profession, "musician")" view="partials/user.html" %}

{% foreach model="page.users" filter="equal(profession, "musician")" view="partials/user.html" %}

{% foreach model="page.users" filter="not_equal(profession, "musician")" view="partials/user.html" %}

{% foreach model="page.users" filter="not_equal(profession, "musician")" view="partials/user.html" %}

{% foreach model="page.users" filter="greater_than(first_name, "Ian")" view="partials/user.html" %}

{% foreach model="page.users" filter="greater_than(first_name, "Ian")" view="partials/user.html" %}

{% foreach model="page.users" filter="less_than_equal(last_name, "Ford")" view="partials/user.html" %}

{% foreach model="page.users" filter="less_than_equal(last_name, "Ford")" view="partials/user.html" %}
