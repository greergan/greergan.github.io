Command line arguments

The generate script accepts multiple command line argument. The first argument must be the path to a JSON configuration file which must be in one of the below forms.

--git path/to/gitaccount.json command line flag is optional.

All addition command line arguments must represent file names which will be matched to specific configuration page objects.

A relative path.
path/to/json_file.json
./path/to/json_file.json
./path/to/json_file.json index.html
./path/to/json_file.json index.html slim.view.html
An absolute path.
/path/to/json_file.json
The "file" protocol.
file:///path/to/json_file.json
The "http" protocol.
http://path/to/json_file.json
The "https" protocol.
https://path/to/json_file.json
Slim example.
slim https://greergan.github.io/slim.generator/generator https://greergan.github.io/models/website.json
Slim example with --git.
slim https://greergan.github.io/slim.generator/generator.ts --git git_configuration.json \
https://greergan.github.io/models/website.json
Deno example.
deno --allow-net --allow-read --allow-write run \
https://greergan.github.io/slim.generator/generator.ts https://greergan.github.io/models/website.json
