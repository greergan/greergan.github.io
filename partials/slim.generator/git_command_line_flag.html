Git command line flag --git

The --git command line flag is optional and specifies the path to a locally stored git configuration file. This configuration cannot be part of the website configuration file if it is stored in a GitHub repository. GitHub has a process that scans repositories looking for GitHub access keys. 

{
    "access_keys": {
        "github": {
            "repositories": {
                "input": {
                    "user": string,
                    "repository": string,
                    "access_key": hash
                }
                "output": {
                    "user": string,
                    "repository": string,
                    "access_key": hash
                }
            }
        }
    }
}
