Configuration options

The website_configuration.json file can contian the following options. Some options are required while others are optional.

Top level configuration properties 
{
    "output_to": path,
    "namespace": path,
    "site": object,
    "copy_to_output": object array,
    "pages": object array
}
Site level configuration properties 
"site": {
    "meta_tags": property array,
    developer_named_property: any
}
copy_to_output level configuration properties 
"copy_to_output": [
    {"path": string},
    {"path": string, "output": string}
],
Page level configuration properties 
"pages": [
    {
        "input_file": path,
        "output_file": path,
        "title": string,
        "meta_tags": property array,
        "external_models": object array
    }
]
External_models level configuration properties 
"external_models": [
    {"path": path },
    {"path": path }
]
