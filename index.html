    Welcome to
    Slim

    A Typescript platform

    Slim is under heavy development. Expect documentation changes as the process proceeds. Generally speaking, this site will evolve along side the development efforts. We try to document a feature and then devlop to the documentation.

    All Typescript code is unique to the Slim project. Absolutely no 3rd party modules are or will be used. Only "import" statements will be supported which means that the Slim project has decided to ignore Javascript "require" statements.

    At some point in the future, after the initial production release of the Slim project, we will be accepting code submissions to be evaluated. We will publish coding standards which will explain the process.

    The Slim project will not allow NPM modules to be installed. There will be a "modules" section of the project which will contain currated code. Once code is accepted, via git merge requests, the code will become part of the Slim git repositories. Only MIT licensed code will be acceptable.

    Slim treats both Typescript and Javascript as first class languages, with Typescript being preferred and required for inclusion in the Slim project.

    Slim.View

    Slim.View is a Typescript HTML rendering engine.

    Slim.Utilities

    Slim.Utilities is a set of reusable Typescript functions.

    Slim.Generator

    Slim.Generator is a Typescript static website generator that uses Slim.View as its rendering engine.
