    Welcome to
    Slim.Filter

    A Typescript JSON filtering engine

    Slim.Filter is a robust filtering system that will be used in Slim.View.

    More information is to come when it is in the development life cycle.

    Boolean filters

     equal

    {%
    foreach model="page.users" filter="equal(active, true)"
	{{first_name}} {{last_name}} {{profession}} {{active}}
    %}

    Dave Gahan musician true
    Harrison Ford actor true
    Ian Astbury musician true
    Robert Smith musician true
    Brian Kernighan programmer true
    Bjarne Stroustrup programmer true

     not_equal

    {% foreach model="page.users" filter="not_equal(active, true)" view="partials/user.html" %}

    Carrie Fisher is a 60 year old actor who is active => false
    Harry Harrison is a 87 year old author who is active => false
    Robert Asprin is a 61 year old author who is active => false
    Dennis Ritchie is a 70 year old programmer who is active => false
