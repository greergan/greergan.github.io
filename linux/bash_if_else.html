bash if examples

  1. create a variable from a command line pipe
  2. then check that the value is valid
  3. if it is not valid then print a message and exit the program

Example 1 liner #1

					
os_name=`uname -s | tr "A-Z" "a-z"`
[[ $os_name != linux ]] && echo "$os_name not supported" && exit

Example 1 liner #2

					
os_name=`uname -s | tr "A-Z" "a-z"`
[ $os_name != linux ] && echo "$os_name not supported" && exit

Example 1 liner #3

					
os_name=`uname -s | tr "A-Z" "a-z"`
if [ $os_name != linux ]; then echo "$os_name not supported"; exit; fi

Example unfolded to traditional syntax

					
os_name=`uname -s | tr "A-Z" "a-z"`
if [ $os_name != linux ]; then
    echo "$os_name not supported"
    exit
fi

Example ready to copy to a file

					
#!/usr/bin/env bash
os_name=`uname -s | tr "A-Z" "a-z"`
if [ $os_name != linux ]; then
    echo "$os_name not supported"
    exit
fi

bash if/else example ready to copy to a file

					
#!/usr/bin/env bash
os_name=`uname -s | tr "A-Z" "a-z"`
if [ $os_name != linux ]; then
    echo "$os_name not supported"
    exit
else
    echo "$os_name supported"
fi
