Example simple case/esac statement

  1. create a variable from a command line pipe
  2. create a simple case statement
  3. write logic for the case
					
#!/usr/bin/env bash
# Find the operating system release name and force it to lower case
release_name=`lsb_release -i | cut -d":" -f2 | sed 's/\s*//' | tr [A-Z] [a-z]`

case $release_name in
    ubuntu) echo "You are running ubuntu";;
esac

Example nested case/esac statements

  1. create a variable from a command line pipe
  2. create a simple case statement
  3. write logic for the case using a nested case/esac statement
						
#!/usr/bin/env bash
# Find the operating system release name and force it to lower case
release_name=`lsb_release -i | cut -d":" -f2 | sed 's/\s*//' | tr [A-Z] [a-z]`

case $release_name in
    ubuntu)
        echo -n "You are running ubuntu"
        release_number=`lsb_release -r | cut -d':' -f2 | sed 's/\s*//'`
        case $release_number in
            20.10) echo " $release_number";;
        esac
    ;;
esac
